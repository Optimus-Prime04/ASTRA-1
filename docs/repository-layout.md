# ASTRA-1 Repository Layout

## Folder Structure

- assets/ → posters, zine pages, branding
- bom/ → bill of materials
- cad/ → CAD exports and source files
- devlogs/ → development progress logs
- docs/ → documentation and safety notes
- firmware/ → ESP32 firmware source code
- media/ → prototype photos and videos
- schematics/ → circuit schematics
- wiring/ → wiring diagrams and pin mapping

## Naming Rules

- Use lowercase file names
- Use hyphens instead of spaces
- Avoid double extensions

Examples:
- astra1-sideview.png
- astra1-assembly.step
- astra1-zine-page.png
